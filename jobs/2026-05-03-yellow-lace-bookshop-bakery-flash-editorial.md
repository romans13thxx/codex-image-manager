## Selection Criteria

- Adult appearance is clear and non-teen-coded.
- Outfit is yellow glossy lace fashion dress with opaque lining, not transparent or bedroom-coded.
- Bookshop/bakery/street cafe scene identity is immediately readable.
- Direct flash is visible without destroying face or hand detail.
- Hands, boots, bag chain, books, and railing geometry are usable.
- No real logo, no readable store text, no watermark, no unintended UI overlay.
- Same outfit, hair, and accessory system stays consistent across the set.

## Failure Fallback

- If age drifts young: increase `adult 22+, mature fashion editorial styling, not teen-coded`.
- If outfit becomes lingerie: increase `opaque lined mini dress worn as street fashion, no bedroom, no transparent fabric`.
- If text appears: increase `pseudo text marks only, no readable letters, no real brands`.
- If hands or bag chain deform: reduce active props to one prop and add `natural hands, five fingers, bag chain separate from fingers`.
- If scene becomes generic: restate exact background anchor, such as `red painted bakery window grid` or `dark wood bookshop shelves and mirror`.

## Memory Candidates

- `yellow glossy lace dress + rose-blonde waves + white platform boots + compact-camera flash` as a reusable fashion-editorial outfit anchor.
- `bookshop mirror / red bakery window / black lamppost / cafe gate` as a repeatable scene set for social street-snap images.
