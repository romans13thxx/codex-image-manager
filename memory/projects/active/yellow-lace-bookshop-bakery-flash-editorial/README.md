# yellow-lace-bookshop-bakery-flash-editorial

Project memory for the yellow lace bookshop, bakery window, and street flash editorial set.

## Goal

Turn the user-provided reference images into a reusable real-human fashion photo system: adult doll-like street editorial, yellow glossy lace dress, rose-blonde hair, white platform boots, compact-camera flash, and repeatable bookshop/cafe/street pose grammar.

## Files

- `style-guide.md`: composition, background, person, prop, UI, and failure rules.
- `prompts.md`: reusable prompt variants, variable slots, and a set contact-sheet prompt.
- `lessons.md`: current observations and future iteration notes.
