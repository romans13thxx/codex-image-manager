# Lessons

## 2026-05-03

- This reference set is best treated as a pose-and-location system, not just a character design.
- The strongest repeatable anchors are blue parking paint, black-yellow poles, generic blue/yellow retail wall, hard sun, fill flash, peach-blonde hair, and recurring accessories.
- Keep UI artifacts as an optional layer. Default clean outputs should remove search pills, hashtags, and watermarks.
- Brand drift is a major risk. Use generic retail signage and abstract yellow blocks instead of readable brand names.
- Age drift is a major risk. Always specify adult 22+ and avoid teen-coded phrasing.
