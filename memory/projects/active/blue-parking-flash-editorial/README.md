# blue-parking-flash-editorial

Project memory for the blue parking lot flash editorial set.

## Goal

Turn the provided reference image set into a reusable project style: bright outdoor retail parking backgrounds, hard sun plus direct flash, peach-blonde adult fashion model, playful pose continuity, and props such as poles, shopping carts, phone, and newspaper.

## Files

- `style-guide.md`: composition and visual rules.
- `prompts.md`: reusable prompt variants and variable slots.
- `lessons.md`: current observations and future iteration notes.

## Taxonomy

- topCategory: `real-human`
- subCategory: `outdoor-street`
- scene: `outdoor`
- skinTone: `light`
- appeal: `cute`
- subject: `female`
