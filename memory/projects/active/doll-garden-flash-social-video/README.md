# Doll Garden Flash Social Video Project Memory

## Project Goal

Build a reusable prompt and composition system for dreamy flower-sunroom portraits: pastel blue chiffon styling, transparent floral umbrella, lace/glass/flower cafe background, and compact-camera flash. Generate clean subject images only, with no app UI.

## Current Effective Style

- Real-human stylized portrait, but with fashion-doll makeup and soft porcelain finish.
- High-key direct flash mixed with daylight, slight overexposure, controlled compact-camera grain.
- Layered flower-room environment: glass window, lace curtain, white wrought-iron chair, lace table, mirror, paintings, dense pastel flowers.
- No UI layer. Keep only subject, physical props, background, lighting, and pose.

## Current Bans

- Do not copy the reference person's exact face or identity.
- Do not generate search bars, play buttons, like/comment/share icons, right-side icon stacks, bottom captions, dates, hashtags, usernames, QR codes, app logos, or watermarks.
- Do not make the character childlike; always specify fictional adult woman in her mid-20s.
- Do not let flowers, umbrella ribs, plush toys, or UI cover the face.
- Avoid excessive jewelry and random text.
