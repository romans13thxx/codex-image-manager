# 3D 渲染偏好画像

> 个人口味层。稳定规则请迁到 `permanent/style-library/3d/`。

## 默认审美倾向

- 材质：物理正确 PBR，避免塑料感
- 光照：三点光为底，主光 45° 上方
- 景深：浅景深，主体锐利
- 分辨率感：细节密度高但不堆砌

## 按子类的默认倾向

| 子类 | 偏好场景 | 偏好气质 | 禁忌 |
| --- | --- | --- | --- |
| stylized-character | studio / 极简背景 | cute、elegant | 过写实毛孔 |
| photoreal-portrait | 室内 / studio | luxury、elegant | 塑料皮肤 |
| product-render | 极简背景 | luxury、cool | 杂乱道具 |
| architectural | 建筑外景 | elegant、cool | 漂浮视角 |
| creature-design | 自然 | cool、energetic | 人类化五官 |
| toy-figure | 极简背景 / studio | cute | 真实皮肤质感 |

## 负面默认项

- 禁止同一张图混合"玩偶质感"与"真实皮肤"
- 禁止无理由的 Dutch angle
- 禁止背景过曝吃掉主体

## 候选规则（待验证）

- 尝试：所有 photoreal-portrait 默认 "sub-surface scattering on, pore-level detail off"
- 尝试：所有 product-render 默认 "seamless backdrop + soft rim light"
