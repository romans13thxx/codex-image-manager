# 2D 动漫偏好画像

> 这是"个人口味"层。稳定复现 ≥ 3 次的规则请迁到 `permanent/style-library/2d-anime/`。

## 默认审美倾向

- 线条：干净赛璐璐线，黑线，避免粗描边。
- 配色：高饱和 + 小面积对比色点缀，优先暖色。
- 光影：软光、柔和漫射，禁用硬阴影。
- 眼睛：高光 ≥ 2 处，避免"死鱼眼"。

## 按子类的默认倾向

| 子类 | 偏好场景 | 偏好气质 | 禁忌 |
| --- | --- | --- | --- |
| shonen | 户外 / 街头 | 热血、cool | 软萌化、低饱和 |
| shojo | 室内 / 花园 | 温柔、elegant | 重阴影、暗色调 |
| isekai | 自然 / 奇幻 | 史诗、energetic | 日常化 |
| slice-of-life | 家 / 校园 | cute、治愈 | 夸张特效 |
| idol | 舞台 / 都市 | energetic、elegant | 粗糙线 |
| fashion-illustration | studio / 极简背景 | luxury、elegant | 复杂背景 |

## 负面默认项（每次都默认追加）

- 没有水印、没有 logo、没有字幕
- 没有六指、没有多肢
- 不要 3D 质感或真人质感

- 尝试：所有 shojo 子类默认 "pastel palette + soft bokeh"
